Introducción a Git

Comando principales

  • git init: inicializa un repositorio git
  • git status: muestra el estado de los archivos del repositorio
  • git add: agrega archivos al área de preparado (staging area)
  • git commit -m "message": guarda todos los archivos preparados
  • git log: muestra el historial de 'commits'
  • git diff: muestra las diferencias entre archivos
  • git checkout: permite moverse entre versiones/branches del proyecto
